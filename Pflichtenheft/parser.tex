
\section{Parser}
\label{sec:parser}





Es soll eine nicht-graphische einfache, imperative Sprache.  Die Sprache
soll in \Mist{} so unterstützt werden, daß man textuelle Spezifikationen
eingeben kann, ohne daß man die graphische Darstellung verzichten muß.
Die graphische Darstellung der Zustände wird von \Mist{} berechnet.


Ein Vorschlag für eine konkrete Syntax findet sich in
Abschnitt~\ref{sec:KonkreteSyntax}. Der erste Schritt der Transformation
(in diesem Modul) wird das textuelle Programm geparst und als abstrakter
Syntaxbaum (ohne graphische Plazierung) dargestellt.
